# Web Application Penetration Testing on DVWA

## Project Overview
This project demonstrates penetration testing on DVWA (Damn Vulnerable Web Application) to identify common vulnerabilities.

## Objectives
- Identify web vulnerabilities
- Perform controlled exploitation
- Suggest mitigation strategies

## 🛠 Tools Used
- DVWA
- XAMPP
- Web Browser
- Burp Suite (optional)

##  Methodology
1. Reconnaissance
2. Scanning
3. Exploitation
4. Analysis

##  Vulnerabilities Identified
- SQL Injection
- Cross-Site Scripting (XSS)
- Brute Force Attack

##  Mitigation
- Input validation
- Prepared statements
- CAPTCHA
- Strong passwords
